## Building

First ensure your development environment is up-to-date. 

Next clone this repository and run make:

```shell script
$ make
```
Note: This build is a static website.
    
To deploy application, run

```shell script
$ make build deploy
```
    
To clean your build, run

```shell script
$ make clean
```
